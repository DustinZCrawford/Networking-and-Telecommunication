# Repository Details
A Brief explination of what is contained in this repository, why these things are here, and what each entity contins.
## Overview
The documents contained in this repository are PDF renders of lab excercises I completed as part of my coursework. Each lab was
formatted with Overleaf, a LaTex document creation application, to ensure each lab was formatted as required with questions, answers, details, and
images to display or give examples of what is being discussed in the document. 

These lab documents use skills such as Virtual Machine configuration/utilization, Linux command line practice for file management and network configuration, examining networks and their protocols using Wireshark, coding servers and clients on the same and separate machines in python, and setting up a host network with a custom local IP address and then using that network to connect to local machines.
